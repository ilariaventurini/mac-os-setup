# ⚡️ Homebrew

## Install

Run `install.sh` script:

```bash
cd ~/mac-os-setup/homebrew
bash install.sh
```

It will install:

- brew (or update it)
- node
- exa
- git
- yarn
- firefox
- google-backup-and-sync
- google-chrome
- jdownloader
- libreoffice
- pdfsam-basic
- runjs
- sip
- slack
- sublime-text
- telegram-desktop
- visual-studio-code
- vlc
- whatsapp
